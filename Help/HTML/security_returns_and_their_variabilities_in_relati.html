Security Returns and Their Variabilities In Relation to Each Other and In Relation to the Market

In the ensuing years financial measures have emerged that relate a security to a market index (of which there are many) and the ratio of a securities expected return premium over some benchmark return.

The former measures are alpha and beta that measure the average difference between the return on the security and the return on the market index and the ratio of the volatility of the security return to the volatility of the market index return, respectively.

The latter is a measure of a security’s return premium over some benchmark to risk. It is measured by the ratio of the expected (average) difference between the return on a security and some benchmark alternative, e.g., the rate on a long-term treasury bond, in relation to the variability of that return premium and the relation of that variability to the variability of the return on the market index. This measure, known as the Modigliani 2 measure (1997), is a modification of the Sharpe Ratio (1966), and allows for comparison of this measure among alternative securities.
