Choosing the Forecast Wizard Parameters

To complete the Forecast Wizard Tasks; (1) the user selects which part of the reference period to use as the source of the return distributions, (2) indicates which kind of distribution to use, empirical or estimated normal, (3) the initial values for all securities, and (4) any particular security(ies) changes.

Here the user has chosen the 2nd half of the reference period, and the estimated distribution method, then clicked the Initialize button. The actions generate the selected rates of return to be equal to the default median value for all securities, and calculated the odds of reversion to the median. Note that the vertical red line "cursors" are at the horizontal axis return value corresponding the 50th percentile of the 2nd half distribution. The user could have moved the cursors using the slider in the lower left-hand corner to adjust the expected ror value, then hit the "Initialize" button to fill all the securities with the rates of return corresponding to the selected percentile.
