Utilize

DelesinPPS allows the user to Store, Print, Export to Excel and/or Submit Trade Orders directly to TradeStation (provided the user is a TradeStation customer).

•select the allocation that best satisfies their portfolio return and risk preferences

•With the aid of the DelesinPPS Trade Calculator Wizard

1.translate the dollar allocations assigned to each security into the integer number of shares for each such security based on the most recent available market prices

2.assign left over funds to the selected securities of their choice

3.seamlessly submit the number of shares of each security to TradeStation for trade execution

4.monitor the orders, trades and P&L position for each security in the portfolio and for the portfolio as a whole in preset windows that show the orders, positions and P&L statements provided by TradeStation.
