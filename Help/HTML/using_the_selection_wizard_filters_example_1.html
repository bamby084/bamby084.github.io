Using the Selection Wizard Filters Example 1

The frequency and cumulative frequency distributions illustrate the boundary effects with an orange color for the unqualified securities (i.e.. those falling outside the specified min-max range) and blue for those lying within. The scatter plots show the risk and return values for all the securities using the same color scheme.

If a plot of only the qualified securities is desired, select the appropriate radio button.

For the example shown above, the minimum value for alpha has been set to the 50th percentile, meaning that all securities with an alpha less than the median are disqualified.

The marginal and cumulative distribution charts show the effect of this constraint.

The scatter plot shows a different perspective of the same results.
