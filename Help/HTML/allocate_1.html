Allocate

For the securities selected, users

•set the constraints, if any, on the percent allocation for each security (en-mass or on an individual basis).

•portfolio target rate of return

•the portfolio value at risk percent

•the maximum percent of average daily trading volume a position may take (en masse or on a security by security basis)

•select an initial portfolio allocation from one of the three solution methods:

1.Min Return Variability

2.Min Downside (Semi) Return Variability

3.Max VaR Point obtained as either a Specific Solution corresponding to the above conditions for each of the three solutions, methods, or from the Efficiency Frontiers generated for each solution method.

•Edit the selected portfolio allocation, if desired, by deleting security allocations the user deems too small for consideration and re-balancing the portfolio.
